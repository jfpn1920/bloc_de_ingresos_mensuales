# 👋 ¡Bienvenidos! a mi proyecto bloc de ingresos mensuales
<img src="imagen_presentacion.png" alt="Presentación" width="220" align="left" style="margin-right: 20px; border-radius: 5px;">
<p style="text-align: justify;">
    ¡Bienvenido a este repositorio! Aquí encontrarás la presentación de mi proyecto <b>bloc de ingresos mensuales</b>, una propuesta enfocada en el registro, organización y administración de ingresos económicos mediante diferentes tecnologías y enfoques de programación. Este proyecto fue desarrollado como ejercicio de aprendizaje utilizando <b>pseint, python, java, visual basic, frontend y backend</b>, permitiendo fortalecer conocimientos en lógica de programación, estructuras de datos, desarrollo web, programación cliente-servidor y bases de datos.
</p>
<p style="text-align: justify;">
    A lo largo del proyecto se implementaron diferentes versiones del mismo sistema, comenzando desde aplicaciones de consola hasta una aplicación web completa con persistencia de datos utilizando <b>localstorage</b> y posteriormente almacenamiento permanente mediante <b>xampp y phpmyadmin</b>. De esta manera, el proyecto permite comprender la evolución progresiva de un sistema de gestión de ingresos desde algoritmos básicos hasta una solución full stack funcional.
</p>

### ✨ Características del proyecto
* Gestión organizada de ingresos mensuales.
* Desarrollo progresivo del mismo sistema en múltiples lenguajes de programación.
* Aplicaciones de consola e interfaz web funcional.
* Almacenamiento local utilizando <b>localstorage</b>.
* Integración completa entre <b>frontend y backend</b>.
* Implementación de base de datos con <b>xampp y phpmyadmin</b>.
* Operaciones crud funcionales (crear, leer, actualizar y eliminar ingresos).
* Actualización dinámica de registros en tiempo real.
* Cálculo automático del total mensual de ingresos.
* Visualización detallada de ingresos registrados.
* Vista previa de cada ingreso.
* Edición y eliminación dinámica de registros.
* Tablero interactivo de ingresos.
* Clasificación de ingresos por categorías.
* Manipulación dinámica del dom con javascript.
* Comunicación entre frontend y backend mediante <b>fetch api</b>.
* Manejo de datos en formato <b>json</b>.
* Persistencia permanente mediante <b>mysql</b>.
* Proyecto desarrollado sin frameworks.
* Organización estructurada de archivos y código.
* Enfoque orientado al aprendizaje progresivo de programación.
